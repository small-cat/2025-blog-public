静态数组，是在声明时就已经确定长度的数组，比如
```
char a[6] = "hello"; // 注意数组尾部的 \0
```
我们可以将 a 这个数组名理解成一个指向数组首地址的指针，比如我们来看如下这几个变量
```
char *pa = a;
char *b = "hello";
```
pa 是一个指针变量，这个指针变量指向了数组首地址。我们来分析一下 pa 和 a 的区别。
a 可以看成是指向数组首地址的指针，这是一个静态数组，是存储在栈上的。而 pa 是指向数组 a 的指针，指针本身是一个栈上的变量，而指向的值是数组 a 的首地址。

此时，无论使用 pa 还是 a 都可以对数组进行遍历和操作。但是对 pa 来说，相当于是一个间接寻址的过程，而 a 是直接寻址。另外，pa 是一个栈上的变量，是可以重新赋值的，比如
```
pa = b;
```
但是 a 这个数组名，虽然理解成指向数组首地址的指针，但是是只读的，不能重新赋值。

再来看 a 和 b 的区别。
b 是一个字符串常量，是一个常量指针，指向的是一个常量字符串。hello 这个字符串保存在常量区中，是 read only 的，我们可以通过 b 来访问这个常量，而 b 指针刚好也指向这个常量字符串的首地址，但是，不能通过
```
b[0] = 'H';
```
这种方式来修改这个字符串。而 a 是可以的，这个是存储在栈上的。

这里需要明确的是数组与指针的区别。
区别就是上面说的，而相同点，一个就是数组名可以看成是指向数组首地址的指针，另一个是作为函数参数的时候，无论数组还是指针都看成是指针来处理。
